CSE 545 Software Security Lab Assignment #1

Overview

Objective:

Simulate various failures in a software system's environment and test whether the software can securely handle the environmental failures.

Lab Hardware & Software

Network:

The VM can be accessed by web browsers on Internet.

Operating Systems:

Windows XP

Tools:

  • Holodeck
  • Microsoft Internet Explorer 6.0

Notes:

  • Students can access their own accounts
  • Instructor and TA can access all the accounts

Lab Tasks Descriptions

  • Simul. 1: Block access to libraries
  • Simul. 2: Block application's registry values
  • Simul. 3: Examine all Common Switches
  • Simul. 4: Force Application to Operate in Low Memory, Disk-space Conditions
  • Simul. 5: Find Any Other Vulnerabilities in IE6 Using Holodeck

Deadline

Monday, Sept. 18, 2012

Grading

  • You are required to document how you have carried out each of these five simulations and make sure the results can be repeated by following your documentation
  • For first two simulations, your grade will be determined whether you can achieve the goals
  • For the other three, your grade will be determined based on how 'creative' your work is and the results you have achieved
  • Submit your report through MyASU assignment link, no later than Monday, Sept. 18, 2012
