CSE 468/598: Computer Network Security Labs

Overview

Objective:

Setup network servers and security features.

Preparations:

Following the instructions on What you need to do in the first week of the semester before doing your course projects in the project websites.

Our TA will give an introductory demo for the virtual lab (vlab.asu.edu) during the first week of the semester.

Virtual machines' connection: V1 <--------> V2 <--------> V3

V-Lab Step-by-Step Guide (PDF) V-Lab Step-by-Step Guide (PPT)

Demo requirements:

Demo will be performed in the same week or the week after the project due during the TA lab hour. We will post the demo schedule online

Project submission:

Please refer to the project requirements described in course website http://dj.eas.asu.edu/courses/CSE468-598/2011fall//Projects.html.

Tasks:

Setup Apache Server

  • Download and install Apache package
  • Enable the webserver on v3 and test it from v2
  • Install and configure the apache server to establish your web service
  • Create a sample webpage on your apache server
  • In report, list all steps you have done on this project and show what configuration you have changed

Setup DNS server

  • Download and install the DNS server
  • Modify the configuration file and make the DNS service work well
  • Create the domain name entries for your web server, DNS server, and your client VMs
  • In the report, list each step you did and what you changed in which configuration file
